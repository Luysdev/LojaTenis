Projeto Loja de Tenis
---------------------

Desenvolvi uma loja virtual em PHP, incorporando recursos cruciais como gestão de produtos, carrinho de compras e finalização de pedidos. Essa experiência foi fundamental para expandir minhas habilidades em manipulação de dados, interação com bancos de dados e garantia de segurança web. Reflete meu compromisso com o aprendizado prático e a busca constante pela melhoria no desenvolvimento web. O programa não focou profundamente na estilização, pois seu principal propósito era servir como uma base para compreender e aplicar a arquitetura MVC e boas práticas durante o processo de desenvolvimento. A ênfase foi dada à implementação dos principais requisitos do sistema, consolidando assim uma base sólida para futuras expansões e aprimoramentos.### Skills
